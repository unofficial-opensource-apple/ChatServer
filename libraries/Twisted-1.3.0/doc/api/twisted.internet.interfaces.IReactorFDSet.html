Class IReactorFDSet

Interface --+
            |
           IReactorFDSet

Implement me to be able to use FileDescriptor type resources.

This assumes that your main-loop uses UNIX-style numeric file descriptors (or at least similarly opaque IDs returned from a .fileno() method)
