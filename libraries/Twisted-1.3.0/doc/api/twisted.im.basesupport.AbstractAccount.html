            |
           AbstractAccount

Known Subclasses:
IRCAccount, PBAccount, TOCAccount

Base class for Accounts.

I am the start of an implementation of IAccount, I implement isOnline and most of logOn, though you'll need to implement _startLogOn in a subclass.
